.. _installation:

Installation
============

You can install serial-dashboard using pip. You will need the dependencies below installed. ::

	$ pip install --upgrade serial-dashboard


Dependencies
------------

serial-dashboard has the following dependencies.

- Python 3.7 or newer
- NumPy_
- Pandas_
- Bokeh_ ≥ 2.3.2
- PySerial_ ≥ 3.5

.. _NumPy: http://www.numpy.org/
.. _Pandas: http://pandas.pydata.org/
.. _Bokeh: https://docs.bokeh.org/
.. _PySerial: https://pyserial.readthedocs.io/
