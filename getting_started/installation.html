Installation

You can install serial-dashboard using pip. You will need the dependencies below installed.

$ pip install --upgrade serial-dashboard

Dependencies

serial-dashboard has the following dependencies.

  • Python 3.7 or newer

  • NumPy

  • Pandas

  • Bokeh ≥ 2.3.2

  • PySerial ≥ 3.5
